# 1. Purpose

This document defines the typography standards for Gym ERP Version 1.0 to ensure readability, consistency, and a premium visual experience.

---

# 2. Font Family

The application shall use:

**Primary Font:** Inter

Reasons:

* Modern appearance
* Excellent readability
* Professional look
* Optimized for desktop applications

---

# 3. Font Weights

The following font weights shall be used:

* Light (300)
* Regular (400)
* Medium (500)
* SemiBold (600)
* Bold (700)

---

# 4. Heading Hierarchy

| Level | Usage                  |
| ----- | ---------------------- |
| H1    | Main page titles       |
| H2    | Section headings       |
| H3    | Card titles            |
| H4    | Small section headings |

The hierarchy shall remain consistent throughout the application.

---

# 5. Body Text

Body text shall:

* Be easy to read.
* Maintain consistent spacing.
* Use Regular (400) font weight.
* Prioritize clarity over decoration.

---

# 6. Tables

Table text shall:

* Be compact but readable.
* Use Medium weight for headers.
* Keep row spacing consistent.

---

# 7. Forms

Forms shall use:

* Clear field labels.
* Readable placeholder text.
* Consistent input sizing.
* Clear validation messages.

---

# 8. Buttons

Button text shall:

* Be centered.
* Use Medium (500) font weight.
* Remain readable at all sizes.

---

# 9. Cards

Cards shall use:

* Bold titles.
* Regular descriptions.
* Consistent spacing between text elements.

---

# 10. Line Height & Spacing

Typography shall maintain:

* Comfortable line height.
* Consistent paragraph spacing.
* Proper spacing between headings and content.

---

# 11. Accessibility

Typography shall:

* Maintain sufficient contrast.
* Avoid excessively small font sizes.
* Remain readable on supported screen resolutions.

---

# 12. Conclusion

The typography standards defined in this document shall be applied consistently across Gym ERP to provide a clean, modern, and professional user experience.
