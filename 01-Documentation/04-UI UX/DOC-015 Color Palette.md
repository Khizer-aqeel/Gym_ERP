# 1. Purpose

This document defines the official color palette for Gym ERP Version 1.0. The selected colors are intended to create a premium, modern, and professional appearance suitable for commercial software.

---

# 2. Design Philosophy

The color palette shall:

* Look premium.
* Reduce eye strain.
* Maintain excellent readability.
* Support both Light Mode and Dark Mode.
* Present a consistent visual identity.

---

# 3. Primary Brand Color

**Primary Color:** Emerald Green

Purpose:

* Primary buttons
* Active navigation items
* Important actions
* Progress indicators

---

# 4. Secondary Color

**Secondary Color:** Deep Blue

Purpose:

* Information cards
* Secondary actions
* Charts
* Supporting highlights

---

# 5. Accent Color

**Accent Color:** Amber

Purpose:

* Notifications
* Warnings
* Important highlights
* Attention indicators

---

# 6. Success Color

Used for:

* Successful operations
* Completed payments
* Active memberships
* Confirmation messages

---

# 7. Warning Color

Used for:

* Memberships nearing expiry
* Equipment maintenance reminders
* General warnings

---

# 8. Error Color

Used for:

* Validation errors
* Failed operations
* Critical alerts

---

# 9. Neutral Colors

Neutral colors shall be used for:

* Backgrounds
* Borders
* Dividers
* Tables
* Cards
* Secondary text

---

# 10. Light Theme

The Light Theme shall use:

* Bright backgrounds
* Dark text
* Soft shadows
* Light borders

The interface shall remain clean and spacious.

---

# 11. Dark Theme

The Dark Theme shall use:

* Dark gray backgrounds
* High-contrast text
* Soft elevation
* Consistent accent colors

The interface shall avoid pure black to reduce eye strain.

---

# 12. Color Usage Rules

* Do not use more than one primary action color on a screen.
* Maintain consistent color meaning throughout the application.
* Avoid excessive use of bright colors.
* Ensure all text meets accessibility contrast standards.

---

# 13. Conclusion

The color palette defined in this document shall be used consistently across Gym ERP to establish a recognizable, professional, and visually appealing brand identity.
